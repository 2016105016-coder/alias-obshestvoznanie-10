# Элиас: обществознание 10

Android-приложение для игры в «Элиас» по обществознанию для 10 класса.

## Что есть внутри

- 1000 карточек-понятий без подсказок и фраз вроде «пример из жизни».
- Темы: обществознание, политология, право, экономика, психология, социальная стратификация, культура, демография, международные отношения, цифровое общество, научные методы.
- 2–8 команд.
- 600 вариантов названий команд.
- Победный порог настраивается вручную от 3 до 100 угаданных слов.
- 4 уровня: лёгкий, средний, сложный, микс 1000.
- Таймер раунда.
- Объёмные 3D-кнопки.
- Программные звуковые эффекты для старта, паузы, продолжения, верного ответа, пропуска, предупреждения таймера, победы и других действий.
- Победный экран с надписью «РЕСПЕКТ ОТ САН САНЫЧА!».
- Unit-тесты для проверки словаря, победного порога, названий команд и звуковых вариантов.
- GitHub Actions для сборки APK.

## Как собрать APK через GitHub

1. Создайте новый репозиторий на GitHub.
2. Загрузите содержимое этого архива в корень репозитория.
3. Откройте вкладку **Actions**.
4. Запустите workflow **Build Android APK** вручную через **Run workflow** или сделайте push в ветку `main`.
5. После сборки откройте завершённый workflow и скачайте artifact `alias-obshestvoznanie-10-debug-apk`.
6. Внутри будет файл `app-debug.apk`.

## Как открыть в Android Studio

1. Откройте Android Studio.
2. Выберите **Open**.
3. Укажите папку проекта.
4. Дождитесь синхронизации Gradle.
5. Запустите приложение на эмуляторе или телефоне.

## Где править словарь

Словарь находится в файле:

`app/src/main/java/ru/school/aliassociety/DeckFactory.java`

Главный ручной список понятий расположен в массиве `BASE_TERMS`. После него приложение добирает словарь до 1000 карточек генерацией обществоведческих словосочетаний.

## Где править интерфейс

Основной интерфейс находится в файле:

`app/src/main/java/ru/school/aliassociety/MainActivity.java`
